SCROLL
  ⌘+shift+h                весь скроллбэк в less
  ⌘+shift+g                вывод последней команды в pager

TABS
  ⌘+t                      новая вкладка
  ⌘+w                      закрыть вкладку/окно
  ⇧+⌘+]  /  ⇧+⌘+[          следующая / предыдущая вкладка
  ⌘+1 .. ⌘+9               перейти к вкладке N

WINDOWS
  ⌘+enter                  новое окно
  shift+⌘+d                закрыть окно
  ⌘+r                      ресайз окна интерактивно

OS WINDOWS
  ⌘+n                      новое OS окно
  ⌘+,                      открыть kitty.conf

FONT
  ⌘+= / ⌘+-                больше / меньше
  ⌘+0                      сброс размера

BUFFER
  ⌘+shift+v                вставить из selection
  ⌘+o                      передать selection в программу

OTHER
  ⌘+shift+,                перезагрузить конфиг
  opt+⌘+,                  debug (текущий конфиг)
  ctrl+⌘+space             ввод unicode
  ⌘+e                      открыть URL 
  kitty +kitten themes     выбрать тему
